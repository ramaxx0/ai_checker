How Password Strength Works

This tool combines rule-based checks and an AI-style composite score to assess password strength. It evaluates length, character variety, and common patterns while estimating entropy and time-to-crack under typical attack assumptions.

What Affects Strength

  • Length: Longer passwords resist brute force exponentially.
  • Variety: Mixing upper/lowercase, numbers, and symbols increases the search space.
  • Avoid Patterns: Common words, keyboard sequences, and repeated characters reduce security.
  • Uniqueness: Do not reuse passwords across accounts.

Entropy

Entropy estimates the unpredictability of a password in bits. Each bit doubles the attacker’s work. This tool estimates entropy based on the character set actually used.

Time to Crack

Estimated cracking time is derived from entropy and assumes high-speed offline attacks. Real-world results vary with attacker capabilities and protective measures like rate limiting and hashing.
